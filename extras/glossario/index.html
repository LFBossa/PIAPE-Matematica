    Teoria dos conjuntos

    Pertence

    • Como se usa:
    • Como se lê: pertence à
    • Sinônimos:
      • elemento pertence ao conjunto
      • é elemento de

    Não pertence

    • Como se usa:
    • Como se lê: não pertence à
    • Sinônimos:
      • elemento não pertence ao conjunto
      • não é elemento de

    Contido

    • Como se usa:
    • Como se lê: está contido em
    • Sinônimos:
      • é subconjunto de

    Diferença entre e

    • também é conhecido como estritamente contido: utilizado para dizer que um conjunto está contido mas não é igual
    • também é conhecido como contido ou igual: utilizado para dizer que um conjunto está contido e pode ser igual

    Quando estiver lendo um livro, veja qual convenção o autor utiliza.

    Não contido

    • Como se usa:
    • Como se lê: não está contido em
    • Sinônimos:
      • não é subconjunto de

    Contido mas diferente

    • Como se usa:
    • Como se lê: está contido mas é diferente de
    • Sinônimos:
      • é subconjunto próprio de

    Quando o autor usa o símbolo para representar a relação de contido, ele usa e como sinônimos.

    União

    • Como se usa:
    • Como se lê: união com
    • Sinônimos:
      • a união do conjunto com o conjunto

    Interseção

    • Como se usa:
    • Como se lê: interseção com
    • Sinônimos:
      • a interseção do conjunto com o conjunto
