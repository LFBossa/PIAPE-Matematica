# Inequação produto

Quando temos um produto 
$$a(x)\cdot b(x)  >0$$

Podemos fazer uma análise do sinal de $a(x)$ e $b(x)$. Quando ambos tiverem o mesmo sinal, o produto será positivo. Quando ambas tiverem sinais distintos, o produto será negativo. 

**Exemplo**

$$(3x -6)*(-2x + 4) > 0$$
